# 🔍 AI Research Assistant

An AI-powered research assistant that takes a user query, searches the web, and generates structured, human-like answers using an LLM.

---

## 🚀 Features

- Web search using Tavily API
- AI-powered summarization using OpenAI
- Agent-based architecture (orchestrator + tools)
- Clean Streamlit UI
- Structured, readable research output with sources

---

## 🧠 Architecture

User Query  
→ Agent Orchestrator  
→ Web Search Tool (Tavily)  
→ LLM Processing (OpenAI)  
→ Structured Answer  
→ Streamlit UI

---

## 🛠️ Tech Stack

- Python
- Streamlit
- OpenAI API
- Tavily API
- dotenv
